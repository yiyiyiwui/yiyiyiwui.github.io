## 🏗️ 核心实战方向

这里不聊乏味的定义，只记录“怎么做”和“为什么这么做”：

### 1. 微服务生态整合 (Spring Cloud)
* **服务治理**：记录 Nacos 作为注册中心与配置中心的深度整合，以及 Feign 接口调优实践。
* **流量防护**：在高并发场景下，如何通过 Sentinel 或分布式限流保障系统不被瞬时流量冲垮。
* **链路追踪**：复杂微服务环境下的问题排查思路与监控方案。

### 2. 高并发与消息驱动 (MQ & Cache)
* **告警长城**：基于百万级并发场景，利用 Redis 位图（Bitmap）或布隆过滤器优化查询性能的方案。
* **异步解耦**：Kafka 与 MQTT 的整合实战，特别是针对物联网（IoT）设备数据的实时接入与处理。
* **缓存策略**：解决缓存击穿、雪崩的实战代码片段，以及 Spring Cache 的深度自定义。

### 3. 数据持久化与复杂查询
* **多库共存**：MySQL 与 PostgreSQL 的选型思考及 Java 多数据源整合方案。
* **检索增强**：Elasticsearch 在海量日志检索与全文本搜索中的整合技巧。
* **ORM 提效**：MybatisPlus 的高级封装，如何减少重复代码，提升开发效率。
