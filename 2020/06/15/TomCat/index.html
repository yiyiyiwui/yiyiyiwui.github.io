TomCat

发表于 2020-06-15 更新于 2023-07-01 分类于 Java

什么是Tomcat？

前端的静态资源和后端的动态资源都要存到一个web服务器中，而你要通过网络访问的话，需要把这些资源放在一个软件内来访问， 这个软件就是Tomcat，Tomcat也被称为web容器，Servlet容器，Servlet程序需要依赖Tomcat才能运行，目前用的是Tomcat9版本

java代码怎么被外界访问？

一个java代码要被外界访问到，必须要遵循Servlet规范，然后放到Tomcat容器中运行，不遵循这个规范的话，Tomcat不认识你这个方法名，就会访问不到

浏览器和服务器之间传输数据怎么才能看懂？

通过HTTP协议，超文本传输协议，规定了浏览器和数据库之间传输数据的格式 ，浏览向服务器发送请求，服务器再响应给浏览器

浏览器怎么通过HTTP向服务器发送请求？

请求行：请求方式，资源路径，协议

请求头：浏览器告诉服务器端，自身的一些信息

请求体：POST请求,存放请求参数

三层架构为什么要实现个接口？

表示层调用业务层，如果业务层代码变了，那么表示层也要跟着改变，所以规范一个接口，业务层实现这个接口，到时候如果业务层改了代码，就不会影响到表示层了。nice！
